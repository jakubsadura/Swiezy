This folder contains files related to the creation of installers. 

'mpi' extension are specific to the InstallJammer tool (http://installjammer.net/).

Check the wiki page (HowToMakeGimiasRelease) to create a GIMIAS installer.

In short:
- copy the mpi file in a folder next to the 'Release' one (the mpi refers to '../Release')
- double click on the mpi
- update the version number
- check the list of files, sort added/reñoved ones
- create the installer
- comit the changes to the mpi file
